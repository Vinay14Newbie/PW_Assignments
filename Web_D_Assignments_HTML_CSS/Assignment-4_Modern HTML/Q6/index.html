Difference between HTML & HTML5

Points HTML HTML5
Document type declaration (DOCTYPE) It requires complex and specific DOCTYPE declaration Simple declaration !DOCTYPE html
Semantics Lacked in semantic tags Introduce new semantic tags for better webpage structure
Audio and Video Required third party plug-ins This supports img and video tags
Accessibility Accessibility features were limited Improve Accessibility
Local Storage Depended on cookies for client-side Storage Introduced 'local Storage' for more efficient client-side Storage
